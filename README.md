## SQL Exercise - Section 1
This exercise aims to further familiarise myself with SQL queries that includes aggregation and window function, which is useful in creating datamarts.


## NY Taxi Data - Section 2
This projects demonstrates an end to end ETL data pipeline using pure python with a focus database **performance optimization**. It processes the NY Green Taxi Data with day of week filtering logic, incremental appends to database with performance optimization such as **partitioning and indexing**, automated aggregation to create datamarts in a form of local CSV, and logging. 

This project is a continuation of my 1st project, approximately after 1 week of focused learning for SQL. 

### Tech Stack

- **Language**: Python 3.12.10  
- **Data Processing**: pandas  
- **Database**: PostgreSQL  
- **Containerization**: Docker & Docker Compose  
- **Logging**: File-based logging  

### Data Source
- **NY Taxi Green Data**
